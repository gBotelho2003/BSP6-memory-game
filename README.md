# BSP6-memory-game
BSP6 project about induced stress on user performance when playing memory games
## 3 Versions:

### Base Version:
Simple memory game where the players objectif is to select the numbers on the grid by order. At the very start the whole grid will be 
briefly flashed. The user has a certain time to finish the level, the points as scored as follow for every wrong selection negative points and plus points
for correct selection as well as sequences and time saved.

### Version 2: 
This has every feature from the base version plus a subtle dark pattern design to inflict pressure and stress upon the player. First there will be a 
display of a leaderboard containing high scores to make the player feel he needs to do better and to add to the pressure when the timer reaches the last 3 seconds
red borders will start blinking around the grid to create urgency.

### Version 3: 
Will contain everything from version 2 plus an hint button that will have an assistent bot helping the player with their next move, by pressing the button the player loses the sequence if he was on one and the button will have a cool down time to prevent spam.
